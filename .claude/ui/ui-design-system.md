# UI DESIGN SYSTEM
# Irie Fishmongers Marketplace

Version: 1.0

This document is the authoritative UI/UX design system for the Irie Fishmongers platform.

All generated screens, components, pages, mobile applications, dashboards, emails, notifications, PDFs, and reports must follow these standards.

---

# DESIGN PHILOSOPHY

The platform must feel:

- Fresh
- Local
- Jamaican
- Trustworthy
- Professional
- Modern
- Vibrant
- Mobile First

The design should communicate:

"Fresh seafood from Jamaica's waters to your table."

Avoid:

- Generic corporate styling
- Dark fintech aesthetics
- Heavy industrial themes
- Overly complex interfaces

---

# BRAND FOUNDATION

Use the official Irie Fishmongers brand guide.

The logo is the primary brand asset.

The logo must appear in:

- Website Header
- Mobile Splash Screen
- Login Screens
- Invoices
- Email Templates
- Vendor Portal
- Driver App
- Admin Dashboard

Never:

- Stretch logo
- Recolor logo
- Rotate logo
- Add effects
- Remove elements
