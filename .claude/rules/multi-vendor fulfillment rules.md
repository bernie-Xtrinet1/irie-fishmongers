# Multi-Vendor Fulfillment

The platform shall support aggregation of inventory from multiple vendors to fulfill a single customer order.

Example:

Customer requests:

50 lbs Snapper

Available inventory:

Vendor A = 20 lbs
Vendor B = 15 lbs
Vendor C = 25 lbs

System allocation:

Vendor A = 20 lbs
Vendor B = 15 lbs
Vendor C = 15 lbs

Total Fulfilled = 50 lbs
