# Mobile Application Rules

## Technology Stack

Required:

- React Native
- Expo
- TypeScript

---

## Applications

Build:

- Customer App
- Vendor App
- Driver App

---

## Mobile UX

Optimize for:

- One-handed usage
- Slow mobile networks
- Battery efficiency

---

## Offline Support

Required:

- cached data
- retry queues
- offline notifications

---



## Navigation

Use:

React Navigation

---

## Forms

Use:

React Hook Form

---

## Notifications

Support:

- push notifications
- order updates
- delivery updates

---

## Device Features

Allow:

- camera
- GPS
- image uploads

Request permissions only when needed.

---

## Performance

Target:

60 FPS

Avoid:

- heavy animations
- excessive API calls

---

## Accessibility

Support:

- screen readers
- large text sizes
- color contrast requirements
