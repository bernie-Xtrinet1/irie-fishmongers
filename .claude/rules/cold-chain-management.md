# Cold Chain Management

## Purpose

Maintain safe seafood temperatures throughout the supply chain.

## Temperature Standards

Fresh Seafood

0°C to 4°C

Frozen Seafood

-18°C or below

## Temperature Monitoring Points

The system shall record temperatures at:

1. Vendor storage
2. Pickup
3. Distribution hub arrival
4. Distribution hub departure
5. Customer delivery

## Data Capture Requirements

Each temperature reading must include:

- Timestamp
- GPS location
- User ID
- Order ID
- Batch ID
- Photo evidence

## Temperature Violations

If temperature exceeds limits:

Status = BREACH

Actions:

- Notify Operations
- Notify Quality Team
- Flag Order
- Record Incident

## IoT Sensor Support

Future refrigerated vehicles shall support:

- Bluetooth sensors
- Cellular sensors
- Real-time telemetry

## Audit Trail

Temperature records must never be deleted.
