# MARKETPLACE MODES
# Irie Fishmongers

Version: 1.0

---

# PURPOSE

The marketplace supports two purchasing modes.

Mode 1

Customer Selected Vendor

Mode 2

Marketplace Fulfillment

The platform must support both.

---

# MODE 1

CUSTOMER SELECTED VENDOR

Customer chooses:

- Product
- Vendor

Customer sees:

- Vendor Name
- Vendor Tier
- Vendor Rating
- Vendor Location
- Compliance Status

---

# MODE 2

MARKETPLACE FULFILLMENT

Customer chooses:

- Product

Platform chooses:

- Vendor

Selection performed automatically.

Customer sees:

Fulfilled by Irie Fishmongers

Best Available Vendor Selected

---

# DEFAULT IMPLEMENTATION

Phase 1

Customer Selected Vendor

Enabled

Marketplace Fulfillment

Disabled

---

Phase 2

Customer Selected Vendor

Enabled

Marketplace Fulfillment

Optional

---

Phase 3

Both Modes Fully Supported

---

# CLAUDE EXECUTION RULE

All marketplace workflows must support both modes.

Mode must be configurable.

Do not hardcode mode behavior.
